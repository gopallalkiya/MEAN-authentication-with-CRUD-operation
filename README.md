
# MEAN Stack CRUD Application with Authentication
MEAN Stack App with Authentication


## Tech Stack

**Client:** Angular, Bootstrap

**Server:** Node, Express, MongoDB

  
## Installation

Clone Repository

Inside Client Folder: Start Client 
```bash
  npm install 
  npm start
```

Inside Server Folder: Start Server 
```bash
  npm install 
  node server.js
```
